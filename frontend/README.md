MedMesh frontend (HTML + CSS)

Quick start:

1. Open `frontend/index.html` in your browser.
2. The UI is static and uses a CSS-only emergency modal. The "Emergency Alert" button toggles the alert overlay.

Notes:
- This is a basic, client-side scaffold (no backend). Use the form markup to hook up APIs later.
- Files added: `index.html`, `styles.css`.
